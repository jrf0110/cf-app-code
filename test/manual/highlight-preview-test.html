Code Highlighting Test

A lot of times, I just want to publish some brain dump HTML generated by markdown. It'd be nice to have an app run on some URL pattern that will automatically add syntax highlighting to my code blocks as well as fix indentation problems.

For instance, the following code block is written using the HTML document's indentation, however, those spaces aren't carried along into the pre-formatting:


      var foo = 'bar'
      if (foo <= 'bar') {
        console.log('baz', foo)
      }

I'd also like to avoid adding a pre tag altogether. That's a plugin option to fix that.

var foo = 'bar' if (foo <= 'bar') { console.log('baz', foo) }

Already highlighted, ignore

Anything that's already highlighted should be ignored.


      var foo = 'bar'
      if (foo <= 'bar') {
        console.log('baz', foo)
      }
